# Приклади Markdown

Цей файл містить приклади синтаксису Markdown, який можна використовувати у навчальному ресурсі.

## Заголовки

```md
# H1
## H2
### H3
```

## Текстове форматування

```md
**Жирний текст**, *курсив*, ~~закреслений текст~~
```

**Жирний текст**, *курсив*, ~~закреслений текст~~

## Списки

```md
- Пункт 1
- Пункт 2
  - Підпункт 2.1

1. Нумерований пункт 1
2. Нумерований пункт 2
```
